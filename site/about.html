BlueBikes Visualization for Year of 2019

About

This site is my final project for the Spring 2020 section of NEU's Bostonography. As part of this project, I aggregated data from multiple different sources and correlated them based on location and time. The main source of information was the BlueBikes usage data, to which I correlated with weather data from NOAA, tree cover data from BARI, and data from the Boston zoning laws. A uniform sampling of trips from 2019 are loaded into the project which uses the D3.js and Crossfilter libraries to allow users to explore the data visually.

The source code for the project is made freely available on GitHub. This includes the data fetching and manipulation code, this site's source code, and the source code for my report.
